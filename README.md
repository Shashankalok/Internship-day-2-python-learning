# Internship-day-2-python-learning
Day 2 of my internship focused on strengthening Python fundamentals, including conditionals, loops, and functions. Practiced problem-solving and built projects such as a calculator and a basic to-do list manager.

Simple Calculator.py (Summary)
This Python program is a simple menu-driven calculator that allows the user to perform basic arithmetic operations such as addition, subtraction, multiplication, and division. First, it displays a list of options for the user to choose from. Based on the user’s input, the program asks for two numbers and checks whether the inputs are valid using a validation condition. If the inputs are correct, it converts them into integers and performs the selected operation, then displays the result. If the user enters invalid numbers or an incorrect operation, the program shows an appropriate error message. Overall, this program demonstrates the use of user input, conditional statements, and basic arithmetic operations in Python.

Todolist.py
This Python program is a graphical to-do list application built using the Tkinter library, which allows users to interact with it through a simple window interface instead of the command line. When the program starts, it creates a main window with a title, size, and background color, and applies styling to make the interface look clean and modern. Inside the window, a frame is used to organize all elements such as the input box, buttons, and task list. The user can type a task into the input field and click the “Add Task” button to add it to the list, which is displayed using a table-like structure called Treeview. A scrollbar is also added so that users can scroll through tasks if the list becomes long. The application provides a “Delete Task” button that allows the user to remove a selected task, and if no task is selected, it shows a warning message. There is also a “Save Tasks” button that stores all the current tasks into a file named `tasks.json` using the JSON format, which helps in saving the data permanently. When the application is opened again, it automatically loads the saved tasks from this file and displays them, ensuring that no data is lost. The program also includes proper layout management so that the interface adjusts when the window is resized. Overall, this application demonstrates how to build a user-friendly GUI, handle user input, manage data, and maintain persistence in Python.
